# Navigation

## Navbar

Create a `navbar.yaml` file in the `src/gatsby-theme-dox` directory.

### navbar.yaml structure

```yaml
# src/gatsby-theme-dox/navbar.yaml
items:      # Navbar Items
  - title:  # Item's title
    link:   # Item's link
  - title:
    link:
```

## Sidebar

Create a `sidebar.yaml` file in the `src/gatsby-theme-dox` directory.

### sidebar.yaml structure

```yaml
# src/gatsby-theme-dox/sidebar.yaml
title:         # Site's title
items:         # Sidebar Items
  - title:     # Items' title
    link:      # Item's link
    items:     # Nested Items (optional)
      - title:
      - link:
  - title:
    link:
```
